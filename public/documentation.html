Request Url Responses
GET /movies Return a list of all movies.
GET /movies/:title Return data by movie title.
GET /movies/:title/genre Return data by movie genre.
GET /movies/director/:name Return data about movie director.
POST /newUser New users registration.
POST /newUser/:id/favorites Add favorite movies.
PUT /newUser/:id/info Update user information.
DELETE /newUser/:id/favorites Remove movies from favorite.
DELETE /newUser Cancel user registration.
